## 电话账单匹配问题
如果有N个电话账单，m个已支付支票(每个支票只能对应于一个账单),设计算法得出哪些电话还没有被支付
### 算法
类似于Merge sort的思路，电话账单和支票都按电话号码排序,电话账单对应的电话号码是so,支票是bj,如果ai<bj则I++,如果ai>bi则j++, 如果两者相则记录到已支付的数组中
